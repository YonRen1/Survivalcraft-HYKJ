# Survivalcraft2-HYKJ

#### 💬模组简介

生存战争荒野科技(中文拼音首字母HYKJ)模组，基于*API1.8*制作。生存战争荒野科技模组始创于2023年10月份，经历多次改版，最终定于现样。

荒野科技模组始终以原始进化为主题，玩家扮演一名海难者被海盗流放荒岛，手无寸铁，只能从原始人开始重新发育。

模组拥有丰富材料、装备、武器、工具，还拥有疾跑、口渴，在这片荒岛生存下去吧！

#### 🔧模组架构

API版本：1.8.2.3
NET版本：9.0

#### 🔴模组开发

1.  使用Termux安装git，克隆本仓库，使用C#shell/MT管理器进行修改(手机开发)
2.  使用git克隆本仓库，用VS等编程程序进行开发(电脑开发)

#### 💢使用说明

1.  HYKJ基于Survivalcraft2API1.8.2.3开发
2.  请积极遵守开源协议
3.  HYKJ水平并不高，欢迎您做出批评指导
