# Survivalcraft2-HYKJ

#### 介绍
生存战争荒野科技模组，基于API1.8制作。模组主题为原始人进化，将人物成长阶段从小到大分为几个阶段。玩家将体验以原始人进化到高科技境界的过程。

#### 软件架构
无过多描述


#### 安装教程

1.  用C#shell就下载压缩包
2.  用VS，git本仓库

#### 使用说明

1.  HYKJ基于Survivalcraft2API1.80开发
2.  使用资源应注明原出处
3.  HYKJ水平并不高，如果您需要，制作组可以提供部分资源如贴图模型等
