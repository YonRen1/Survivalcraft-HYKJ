# Survivalcraft2-HYKJ

#### 介绍
生存战争荒野科技模组

#### 软件架构
荒野科技拥有真实的生存体系


#### 安装教程

1.  使用VS或C#shell
2.  用VS就，git本仓库，用C#shell就下载压缩包
3.  开发完后新建Pull Request等待审核后合并仓库
#### 使用说明

1.  HYKJ基于Survivalcraft2API1.80开发

#### 参与贡献

3.  提交代码
4.  新建 Pull Request
