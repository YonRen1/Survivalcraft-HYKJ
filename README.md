# Survivalcraft2-HYKJ

#### 介绍
生存战争荒野科技模组，基于API1.8.1.1制作。模组主题为原始人进化，将人物成长阶段从小到大分为几个阶段。玩家将体验以原始人进化到高科技境界的过程。

#### 软件架构
API版本：1.8.1.1

#### 安装教程

1.  用C#shell(手机开发)
    用电脑模式打开本仓库网页，点下载zip
2.  用VS等(电脑开发)
    git本仓库

#### 使用说明

1.  HYKJ基于Survivalcraft2API1.8.1.1开发
2.  禁止用于商用，使用资源应注明原出处
3.  HYKJ水平并不高，您可以做出批评指导，主开发者：永仁(3570678237)
